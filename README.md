# ⚡ HackAI – Autonomous Placement Intelligence System

> "A real AI system that predicts, decides, and acts — not just responds"

A production-ready full-stack MERN application powered by Agentic AI that autonomously monitors students, predicts placement outcomes, detects risks, intervenes automatically, and generates job-tailored resumes from real developer profiles.

---

## 🏗️ Tech Stack

| Layer | Technology | Purpose |
|---|---|---|
| Frontend | React 18 + Recharts + React Router v6 | UI, charts, routing |
| Backend | Node.js + Express.js (MVC architecture) | REST API, business logic |
| Database | MongoDB Atlas (Mongoose ODM) | Data persistence |
| AI / LLM | Groq API — Llama 3.3 70B (free, 14,400 req/day) | Resume analysis, chat, interview, tasks, digital twin |
| Job Market Data | JSearch API via RapidAPI | Real jobs from LinkedIn, Indeed, Glassdoor, ZipRecruiter |
| GitHub Data | GitHub REST API (public, no key needed) | Repos, languages, stars, project analysis |
| LeetCode Data | LeetCode GraphQL API (public) | Problems solved, ranking, topics |
| Auth | JWT (JSON Web Tokens) | Secure login, role-based access |
| File Upload | Multer | PDF resume upload |
| PDF Parsing | pdf-parse (with malformed PDF fallback) | Extract text from any PDF |
| Scheduling | node-cron | Daily autonomous agent cycles |
| PDF Export | html2pdf.js (CDN) | Download generated resume as PDF |
| Styling | Custom CSS — dark theme | Responsive UI |


```env

MONGO_URI=your_mongodb_atlas_connection_string
JWT_SECRET=any_random_secret_string
GROQ_API_KEY=gsk_xxxxxxxxxxxx
```


```bash

cd hackai/backend
bash start.sh

# Terminal 2 — Frontend
cd hackai/frontend
npm start

```

### 👨‍🎓 STUDENT FEATURES

---

#### 1. Dashboard (`/dashboard`)
The central hub for every student.

**Shows:**
- Placement probability meter (0–100%) with color coding
- Risk level badge: Ready / At Risk / Unprepared
- AI-generated daily tasks with one-click completion
- Job application tracker with status (Applied → Shortlisted → Interview → Offered → Rejected)

**How it works:**
- On every login, `lastActive` timestamp is updated
- Tasks are generated by Groq LLM daily via cron job
- Alerts are created automatically when risk is detected

---

#### 2. Resume Analyzer (on Dashboard)
Upload a PDF resume and get instant AI analysis.

**Flow:**
1. User uploads PDF → Multer saves to `/uploads/`
2. `pdf-parse` extracts text (with fallback for malformed/Canva PDFs)
3. Text sent to Groq Llama 3.3 70B
4. AI returns structured JSON: `{ skills, weaknesses, projects, summary }`
5. Profile updated in MongoDB
6. Placement probability recalculated immediately

**Handles:** Standard PDFs, Canva exports, Google Docs exports, scanned PDFs (with raw text fallback)

---

#### 3. Job Recommendations (`/jobs`)
Skill-matched job listings sorted by compatibility.

**How matching works:**
```
matchScore = (matched skills / required skills) × 100
```
- Compares student's extracted skills vs job's `skillsRequired`
- Sorted highest match first
- Apply button with duplicate prevention
- Activity score boosted on each application

**Skill Gap Analysis button (per job):**
- Click "🔍 Skill Gap" on any job
- Modal shows: matched skills (green), missing skills (red)
- Match % progress bar
- Learning roadmap: specific course suggestions per missing skill with priority level (High/Medium)

---

#### 4. AI Career Coach (`/coach`)
GPT-powered personal placement mentor.

**How it works:**
- Student profile injected into system prompt: skills, weaknesses, placement probability
- Conversation history maintained (last 6 messages for context)
- Uses Groq Llama 3.3 70B
- Quick prompt buttons: "How to improve resume?", "Tips for DSA?", etc.
- Responses are personalized — not generic

---

#### 5. Mock Interview System (`/interview`)
Full AI-powered technical interview simulation.

**Flow:**
1. Select topic: JavaScript, Python, DSA, System Design, SQL, React, General CS
2. AI generates question based on student's skill profile
3. Student types answer
4. AI evaluates: score (0–10), feedback (2-3 sentences), improvement suggestions
5. Repeat for 5 questions
6. Final score screen with performance summary

**After each session:**
- `InterviewResult` saved to MongoDB with sub-scores
- Student's `interviewScore` updated (rolling average)
- Placement probability recalculated

---

#### 6. Interview Analytics (`/interview-analytics`)
Visual breakdown of all mock interview sessions.

**Shows:**
- Total sessions, average score, technical/communication/confidence averages
- Bar chart: score breakdown by category
- Pie chart: score distribution
- Full session history table with dates and per-category scores

**Sub-scores calculated:**
```
technicalScore    = score × 0.9 + random(0,1)
communicationScore = score × 0.85 + random(0,2)
confidenceScore   = score × 0.95 + random(0,1)
```

---

#### 7. Placement Prediction Engine (`/prediction`)
AI-powered score with trend analysis.

**Formula:**
```
placementScore = (skillScore × 0.4) + (appScore × 0.3) + (interviewScore × 0.3)

skillScore     = min(skills.length × 4, 40)    → max 40 pts
appScore       = min(applications × 6, 30)     → max 30 pts
interviewScore = (avgScore / 10) × 30          → max 30 pts
```

**Also calculates:**
- `confidence` — data completeness score (0–100%)
- `trend` — "up" / "down" / "stable" vs previous stored score

**Shows:**
- Big score number with color (green ≥70, yellow ≥40, red <40)
- Trend arrow ↑ ↓ →
- Score breakdown with individual progress bars
- 7-day trend line chart (Recharts LineChart)

---

#### 8. Digital Twin (`/digital-twin`)
AI-generated dynamic profile — your virtual career identity.

**What it shows:**
- Radar chart: student skills vs ideal benchmark (6 key skills)
- 7-day growth trend line chart
- AI-generated strengths (3 items)
- AI-generated weaknesses (3 items)
- AI-generated recommendations (3 action items)

**How it works:**
- Groq generates strengths/weaknesses/recommendations based on actual profile data
- Radar data compares student skills against: Data Structures, System Design, JavaScript, SQL, Communication, Problem Solving
- Growth trend simulated from current score + trend direction

---

#### 9. Market Insights (`/market`)
Real-time job market intelligence.

**Real job listings:**
- Fetched live from JSearch API (LinkedIn, Indeed, Glassdoor, ZipRecruiter)
- Queries: "software engineer India" + "frontend backend developer India"
- Falls back to 8 curated mock jobs if API unavailable

**Real trending skills:**
- NOT hardcoded — calculated dynamically from actual job descriptions
- Algorithm:
  1. Fetch all live jobs
  2. Count skill frequency across all job descriptions
  3. Normalize to percentage (most-mentioned = 100%)
  4. Save to `TrendingSkill` MongoDB collection
- Updates every time the page loads

**Shows:**
- Skill demand tags with % badges and category colors
- Horizontal bar chart (Recharts BarChart)
- Live job listings with company, role, location, salary, required skills

#### 10. AI Career Intelligence System — Resume Generator (`/generate-resume`)
The most advanced feature. Generates a complete ATS-friendly resume from real developer profiles.

**Input:**
- GitHub username → fetched via GitHub REST API
- LeetCode username → fetched via LeetCode GraphQL API
- LinkedIn summary → user-pasted text (scraping blocked by LinkedIn)
- Target job → searched live from JSearch API

**GitHub data fetched:**
- Name, bio, location, public repos, followers
- Top languages (weighted by stars + frequency)
- Detected frameworks (React, Django, Express, etc. from repo names/descriptions)
- Top 3 projects by stars with: name, description, language, stars, topics
- Project impact tagging: High Impact (≥10 stars) / Moderate (≥3 stars) / Basic

**LeetCode data fetched:**
- Total problems solved (Easy / Medium / Hard breakdown)
- Global ranking
- Top 5 topics by problems solved
- Skill level: Beginner (<100) / Intermediate (100-299) / Advanced (≥300)
- LeetCode strength score (0-100): `(easy×1 + medium×2 + hard×4) / 10`

**Intelligence scores calculated (no AI needed):**
```
resumeScore = (skills × 5, max 30)
            + (projects × 10, max 20)
            + (lcScore × 0.3, max 20)
            + (hasStars ? 10 : 0)
            + (repos ≥ 5 ? 10 : 5)
            + (hasLinkedIn ? 10 : 0)
            → capped at 100

profileCompleteness = GitHub(30) + LeetCode(25) + LinkedIn(20) + 1project(15) + 3projects(+10)
```

**Skill intelligence:**
- Strong skills: present in both GitHub languages AND LeetCode topics
- Weak skills: in GitHub but low activity/no LeetCode backing
- Recommended skills: trending market skills not yet in profile

**AI generates (Groq Llama 3.3 70B, 3 parallel calls):**
1. Full ATS-friendly resume (plain text, no markdown tables)
2. Structured JSON: `{ suggestions[], roadmap[], summary, highlights[] }`
3. Resume sections: Header, Summary, Skills, Projects, Coding Achievements, Education

**Job-aware mode:**
- Search real jobs from JSearch directly in the form
- Select any job as target
- System calculates: `jobMatchScore` (% of required skills you have)
- Lists `missingSkills` for that specific role
- Auto-assigns learning tasks to student dashboard: "📚 Learn Docker to qualify for DevOps Engineer at Zepto"

**Output tabs:**
1. Resume Preview — clean white resume with AI summary banner
2. Suggestions — 5 numbered AI suggestions
3. Roadmap — 4-week color-coded improvement plan
4. Profile Data — GitHub repos, project cards with impact badges, LeetCode stats

**Download PDF** — html2pdf.js generates clean A4 PDF

## 🤖 Autonomous Agent System

### How It Works
The agent runs on two cron schedules and operates completely autonomously:

```
Daily at 7 AM — Agent Cycle (agentCron.js):
  For every student:
    1. Detect risk level (HIGH / MEDIUM / LOW)
    2. HIGH RISK → trigger full intervention
    3. MEDIUM/LOW → generate personalized AI tasks via Groq
    4. Recompute placement prediction score
    5. Log all actions

Daily at 8 AM — Risk Detection (riskCron.js):
  For every student:
    1. Check: skills < 3, applications = 0, inactive > 5 days, interview < 4
    2. HIGH RISK → trigger intervention (tasks + alerts + log)
    3. Report summary to console
```

### Risk Classification Logic
```
HIGH RISK (all 3 conditions):
  skills < 3 AND applications = 0 AND inactive > 5 days
  → Full intervention triggered

MEDIUM RISK (any 2 conditions):
  skills < 5 OR applications < 3 OR interview score < 4
  → AI tasks generated

LOW RISK (any 1 condition):
  → Gentle task nudge
```

### LLM Usage (Groq — Llama 3.3 70B)

|---|---|---|
| Resume Analysis | Extracts skills, weaknesses, projects as JSON | ~600 |

| Digital Twin | Generates strengths, weaknesses, recommendations | ~400 |
| Task Generation | Creates personalized daily tasks | ~400 |

## 📡 Real Data Sources

### 1. JSearch API (RapidAPI)
- **What:** Aggregates real jobs from LinkedIn, Indeed, Glassdoor, ZipRecruiter, Shine, Apna Jobs
- **Used for:** Market Insights page + Resume Generator job search
- **Free tier:** 200 requests/month
- **Sign up:** https://rapidapi.com/letscrape-6bRBa3QguO5/api/jsearch
- **Fallback:** 8 curated mock jobs when key not set

### 2. GitHub REST API
- **What:** Public API, no key required for basic usage
- **Used for:** Resume Generator — repos, languages, stars, project data
- **Rate limit:** 60 req/hour unauthenticated, 5000/hour with `GITHUB_TOKEN`
- **Optional:** Add `GITHUB_TOKEN=your_token` to `.env` for higher limits

### 3. LeetCode GraphQL API
- **What:** Unofficial public API
- **Used for:** Resume Generator — problems solved, ranking, topics
- **No key required**

### 4. Trending Skills
- **What:** Calculated dynamically from real JSearch job descriptions
- **Algorithm:** Count skill keyword frequency → normalize to % → store in MongoDB
- **Updates:** Every time `/api/jobs/trending` is called
